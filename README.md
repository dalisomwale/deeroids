
## Features

- Responsive design
- Portfolio filtering
- Project modals with slideshows
- Lazy loading images
- Contact form
- Mobile-friendly navigation

## Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Font Awesome Icons

## Setup

1. Clone or download the project files
2. Open `index.html` in a web browser
3. For development, use a local server for best performance

## Customization

- Update content in `index.html`
- Modify styles in CSS files
- Add portfolio items by following the existing structure
- Update contact information
